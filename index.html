Sintaxis de selectores

es relativamente sencilla y se basa en la combinación de diferentes elementos y atributos.


Estos son vasior de los Elementos Basicos

Nombres de elementos:Se utilizan los nombres de los elementos HTML (por ejemplo, `p`, `h1`, `div`) para seleccionar todos los elementos de ese tipo

Clases: Se utilizan puntos (.) seguidos del nombre de la clase para seleccionar todos los elementos que tienen esa clase (por ejemplo, `.destacado`).

IDs: Se utilizan símbolos de almohadilla (#) seguidos del nombre del ID para seleccionar el único elemento que tiene ese ID (por ejemplo, `#titulo`)

Combinadores: Se utilizan operadores especiales para combinar selectores y crear reglas más específicas (por ejemplo, `>`, `+`, `~`).

Atributos: Se utilizan corchetes [] para seleccionar elementos que tienen un atributo específico (por ejemplo, `[href]` selecciona todos los elementos que tienen un atributo `href`).
